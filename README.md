# OCR Text Recognition Engine

> Автономный локальный микро-сервис на Node.js, Express и Tesseract.js для локального оптического распознавания текста (OCR) с изображений и документов без отправки данных на сторонние серверы.

---

## 🚀 Архитектура и стек

* **Backend:** Node.js, Express, Multer (обработка загрузки файлов).
* **OCR Движок:** Tesseract.js с локально подключенными языковыми обученными моделями (`rus.traineddata`, `eng.traineddata`).
* **AI Очистка (Groq):** Интеграция с Groq API (`llama3-70b-8192`) для авто-исправления опечаток и распознанного текста.
* **Frontend:** HTML5, Tailwind CSS, Vanilla JS, Panzoom (интерактивный Drag & Drop интерфейс, масштабирование, копирование результата в один клик).

---

## 🛠 Проблема и Решение

* **Проблема:** Перебор текстов из скриншотов, сканов и PDF вручную требует монотонной работы, а публичные OCR-сервисы создают риски утечки конфиденциальных данных.
* **Решение:** Локальный OCR-конвейер: пользователь перетаскивает изображения в браузер, локальный движок Tesseract мгновенно распознает русско- и англоязычный текст прямо на машине пользователя, а нейросеть Llama 3 автоматически исправляет опечатки.

## 📦 Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone [https://github.com/MADNIKE1991/ocr-text-engine.git](https://github.com/MADNIKE1991/ocr-text-engine.git)
   cd ocr-text-engine
   ```

2. Установи необходимые зависимости (скачает все пакеты из `package.json` и развернет локальную папку `node_modules`):
   ```bash
   npm install
   ```

3. Запусти сервер:
   ```bash
   npm start
   ```

4. Открой в браузере:
   ```text
   http://localhost:3644
   ```
